# [Docs](https://github.com/georgegillams/gg-components/tree/main/docs) > Theming

Components can be themed using CSS modules by setting the following variables:

```
--primary-color
--primary-color-dark
--primary-color-invert
--primary-color-light

--checkbox-tick-color
--checkbox-tick-color-invert

--cta-color
--cta-color-dark
--cta-color-invert
--cta-color-light

--link-color
--link-color-hover
--link-color-invert
```
